About this Project

This is my second project, and first semester project.

For this site I designed and coded a website for a fictional museum that was going to open in Bergen. The target audience was kids and young adults interested in science and technology, and also families looking for family friendly fun.

I used mainly HMTL and CSS, but used JavaScript for the navigation menu on phones and tablets.

Technology used:

  • HTML
  • CSS
  • JavaScript
  • Photoshop
